# 안성수 Backend Developer Portfolio

GitHub Pages로 배포할 수 있는 정적 포트폴리오 웹페이지입니다.

## 구성

- `index.html`
- `style.css`
- `assets/profile.png`

## GitHub Pages 배포 방법

1. GitHub에서 새 Repository를 생성합니다.
2. 이 폴더 안의 파일을 그대로 업로드합니다.
3. Repository의 `Settings` → `Pages`로 이동합니다.
4. `Build and deployment`에서 `Deploy from a branch`를 선택합니다.
5. Branch를 `main`, Folder를 `/root`로 선택하고 저장합니다.
6. 잠시 후 `https://깃허브아이디.github.io/레포지토리명/` 주소로 접속할 수 있습니다.
